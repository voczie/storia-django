# Versão do StorIA para Django (Utilizado durante meu projeto de iniciação científica)
Este é uma adaptação para Django de um projeto já existente, feito em Flask, da TAIL (Technology and Artificial Intelligence League), liga acadêmica da Universidade Federal da Paraíba, da qual faço parte da diretoria. O projeto em questão se chama StorIA e o repositório dele pode ser acessado [aqui](https://github.com/TailUFPB/storIA).
